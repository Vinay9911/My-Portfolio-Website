Advanced Retrieval-Augmented Generation (RAG) System with Multi-File and LLM Integration

This project showcases a comprehensive Retrieval-Augmented Generation (RAG) system designed to handle a variety of document types, including PDFs, Word files, TXT files, and URLs. The system extracts text from these sources and uses SentenceTransformer to vectorize the content. It employs FAISS for fast and accurate content retrieval. The application integrates with the LLM Model mixtral-8x7b-32768 provided by Groq for generating contextually relevant answers to user queries. The system also features an interactive chat interface, allowing users to engage with the LLM for dynamic, intelligent responses.

NLP-Driven FAQ Chatbot with Comprehensive User Interaction Analysis

Developed a Flask-based intelligent chatbot that leverages TF-IDF and cosine similarity to efficiently match user queries with answers stored in a JSON-based knowledge base. The chatbot dynamically finds the most relevant answers by comparing user inputs to predefined questions, ensuring accurate responses. Additionally, it features a comprehensive interaction logging system, capturing user queries, chatbot responses, and feedback for continuous improvement. This project highlights skills in Flask, natural language processing, and interaction tracking, making it a valuable addition to any portfolio.
